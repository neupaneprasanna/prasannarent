# User Functionality Checklist

This document provides a comprehensive list of features and functionalities available to a standard user of RentVerse. Use this to verify the website's behavior.

## 1. Public Experience (Unauthenticated)
- [ ] **Home Page**
    - [ ] Dynamic 3D Hero Scene with interactive light tracking.
    - [ ] "Explore Now" navigation.
    - [ ] Category-based filtering.
    - [ ] Trending Rentals Carousel.
    - [ ] Live Platform Stats counter.
    - [ ] Interactive World Map showing rental locations.
    - [ ] **Mobile POV**: Verify grid stacking and touch interaction.
- [ ] **Explore / Search**
    - [ ] AI-powered search bar (with suggestions).
    - [ ] Search listings by title, category, or location.
    - [ ] Realistic 3D Map view with high-res satellite imagery and markers.
- [ ] **Item Details**
    - [ ] View detailed rental information, price units (Hour/Day/Week/Month).
    - [ ] High-quality image gallery.
    - [ ] Host information and verified status.
    - [ ] Reviews and ratings section.
- [ ] **Authentication** 
    - [ ] Sign Up with email, name, and phone.
    - [ ] Login with secure credentials.

## 2. Authenticated User Experience (Dashboard)
- [ ] **Overview**
    - [ ] Personal stats (Total Listings, Active Bookings, Total Earnings).
    - [ ] Revenue Chart (Gross earnings visualization).
    - [ ] Weekly Activity Chart (Views and interactions).
- [ ] **Listings Management**
    - [ ] Create New Listing (Title, Category, Price, Images, Location).
    - [ ] View "My Listings" with status indicators.
    - [ ] Edit/Update existing listings.
- [ ] **Bookings**
    - [ ] View incoming and outgoing bookings.
    - [ ] Track booking status (Pending, Confirmed, Active, Completed, Cancelled).
- [ ] **Earnings**
    - [ ] Track financial totals.
    - [ ] Export earnings data.
- [ ] **Communication & Notifications**
    - [ ] Real-time Messaging system with other users.
    - [ ] Notification feed for platform updates and booking alerts.
- [ ] **Account Settings**
    - [ ] Profile management (Avatar upload, Bio, Personal info).
    - [ ] Password updates and 2FA configuration.
    - [ ] Interest selection for personalized recommendations.

## 3. Interaction Features
- [ ] **Reviews**
    - [ ] Rate listings and leave text reviews.
    - [ ] Upload images as part of the review process.
- [ ] **Engagement**
    - [ ] "Love" (Like) listings to save them (Affects "Popular" rankings).
    - [ ] View count tracking on personal listings.
